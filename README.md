# memphis-v4

Production-first, basic-but-working foundation for Memphis v4.

## Quick start
```bash
npm install
cp .env.example .env
npm run dev
```

## Scripts
- `npm run dev` — run local entrypoint
- `npm run typecheck` — TypeScript checks
- `npm run lint` — ESLint
- `npm run test` — Vitest
- `npm run build` — compile to `dist/`
