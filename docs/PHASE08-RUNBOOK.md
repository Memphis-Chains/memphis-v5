# PHASE 08 Runbook

## Validate tooling
```bash
npm run -s ops:phase08-verify-tooling
```

## Run full smoke pack
```bash
npm run -s ops:phase08-smoke-pack
```

## Run smoke pack with report output
```bash
npm run -s ops:phase08-smoke-pack:report
```

## Cleanup runtime artifacts
```bash
npm run -s ops:phase08-clean
```
